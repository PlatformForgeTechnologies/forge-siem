Roadmap

What we're building

A transparent look at what we've shipped and where Forge SIEM is headed. Updated as priorities evolve.

Q1 2026

Foundation

Completed

Full event pipeline

Agent mTLS transport, ingest, decoder, Sigma rules engine, alert dedup, compliance tagging, and raw archiving — all streaming through Redis.

ClickHouse storage backend

MergeTree tables for events, alerts, and raw logs with TTL-based lifecycle. LowCardinality compression on high-frequency fields.

Sigma detection rules

31 rules across Linux auth, process, FIM, network, Kubernetes, and Windows auth — mapped to MITRE ATT&CK with sub-second alert latency.

Agent with mTLS enrollment

Standalone Go binary for Linux and Windows. Log tailing, FIM, process monitoring, package inventory. Enrollment PKI with CN-bound client certificates.

REST API and analyst dashboard

Session-authenticated API with RBAC. Dashboard with alert triage, agent management, log search, network flows, vulnerability tracking, and case management.

Helm chart deployment

One-command deploy via ArgoCD. Self-hosted and BYO dependency modes. KEDA autoscaling on stream lag. GitHub Actions CI with per-component builds.

Q2 2026

Platform Hardening

Completed

Vulnerability & image scanning

NVD CVE catalog with 2,000+ CVEs, polled every 6 hours. Per-agent CVE matching and Trivy Operator integration for Kubernetes image scanning.

Active response framework

HMAC-signed commands delivered via Redis, RabbitMQ-based response orchestration, and Slack webhook notifications for high-severity alerts.

Admin center

Separate React app for platform administration — user management, billing operations, audit log, and platform settings. Role-gated to admins only.

Billing backend

Database schema, store layer, and API routes for subscription management. Plan widget in the customer dashboard.

Agentless collection

HTTP POST and UDP syslog collector for devices and services that can't run an agent — firewalls, network appliances, cloud-native services.

Q3 2026

Multi-Tenancy & SaaS

In Progress

Multi-tenant architecture

Tenant-scoped data model across all resources. JWT-based API middleware. Per-tenant roles — owner, admin, analyst, viewer.

Enrichment worker

Automated alert enrichment with AbuseIPDB, GreyNoise, VirusTotal, and geolocation. Observable extraction for IPs, hashes, and domains.

Stripe billing & usage metering

Subscription management, daily usage metering from ClickHouse, plan enforcement, and embedded Stripe customer portal.

Onboarding wizard

Signup to first event in under 5 minutes. Account creation, email verification, cloud provider connection, agent install — all guided.

Customer & admin portals

Customer portal for org settings, team management, API keys, and billing. Forge admin portal for tenant operations and system health.

Q4 2026

Cloud & Data Plane

Planned

Sync agent

Outbound-only service in customer's data plane. Rule sync, alert metadata push, and WebSocket query proxy — no inbound ports needed.

AWS BYOC integration

One-click CloudFormation for IAM cross-account role. Helm chart deploys into customer EKS. S3 for raw log archival.

GCP & Azure BYOC

Service account + Workload Identity for GCP. App registration + RBAC for Azure. Cloud-native object storage for each provider.

Forge-hosted launch

platformforge.com goes live — fully managed SIEM with zero-ops deployment. Per-agent and storage-based pricing.

2027

Compliance & Automation

Planned

SOC 2 Type 1

Access control, incident response policy, vendor review, audit log, MFA, encryption at rest — with compliance consultant.

Detection-as-code CI

Customers push Sigma rules to their own git repo. Forge CI validates syntax, tests against sample events, and deploys via sync agent.

SOAR playbooks

Response playbooks triggered from deduped alerts. Visual playbook authoring UI on top of the existing RabbitMQ orchestration layer.

Linode BYOC & ISO 27001

Linode cloud integration for BYOC deployments. ISO 27001 certification following SOC 2 with expanded information security management scope.
